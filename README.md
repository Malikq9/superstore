# Superstore Returns Analysis Project

## Project Description

This project provides an in-depth analysis of product returns at the Superstore, aiming to uncover patterns and insights that could help reduce the volume of returns and enhance customer satisfaction. Utilizing Tableau, we've developed an interactive dashboard that explores various dimensions of returns, including seasonality, product categories, and geographic distribution.

## Tableau Public Workbook

Explore the interactive dashboard on Tableau Public to dive deep into our findings and understand the strategies proposed to minimize returns at the Superstore.
